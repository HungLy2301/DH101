# Accessibility Statement

This site was built by Hung Ly for DH 101.

I have tried to make it accessible by:

• Adding descriptive alt text to all images, including the network visualization (e.g., "Network graph showing relationships between 30 nodes representing apps, companies, and data types in a personal digital ecosystem. Central nodes include Google, Apple, Meta, and Microsoft.")

• Checking all text-background color combinations using the WebAIM Contrast Checker to ensure a minimum contrast ratio of 4.5:1 for normal text

• Replacing all vague link text (such as "click here") with descriptive text (e.g., "View the full network on Kumu")

• Structuring the page with proper heading hierarchy (H1 → H2 → H3) so screen reader users can navigate by headings

• Ensuring the network visualization is accompanied by a full text description for users who cannot see the visual

If you encounter barriers accessing any content on this site, please contact me at ly_h1@denison.edu
